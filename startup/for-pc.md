# PC 环境搭建

### 系统环境要求

> 如无特殊说明，以下软件建议使用最新版本

- [ ] JDK 1.8
- [ ] Eclipse for Enterprise Java Developers
- [ ] [Eclipse 插件安装](https://github.com/orgs/WiseCRM/teams/wisecrm/discussions/3)
- [ ] VSCode
- [ ] Tomcat 8.0+
- [ ] Maven 3.5+
- [ ] GitHub Desktop
- [ ] Nodejs（建议安装）

### 导入项目

1. 通过 `git clone` 或 GitHub Desktop 签出代码
2. 用 Eclipse 导入为 Maven 工程
3. 然后将 `.eclipse/` 内的文件复制到项目目录下
